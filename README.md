# 💳 Credit Card Approval Prediction System

A Machine Learning-based web application that predicts whether a credit card application is likely to be **Approved** or **Rejected** based on applicant information.

The project is developed using **Python**, **Flask**, **Scikit-learn**, **XGBoost**, **HTML**, **CSS**, and **JavaScript**. It features a modern AI-powered interface, an interactive dashboard, confidence score visualization, AI risk analysis, and a responsive user experience.

This project demonstrates the integration of Machine Learning with Full Stack Web Development to solve a real-world financial prediction problem.
