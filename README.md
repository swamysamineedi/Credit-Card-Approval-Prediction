# 💳 Credit Card Approval Prediction System

A **Machine Learning-powered web application** that predicts whether a credit card application is likely to be **Approved** or **Rejected** based on an applicant's financial and personal information.

Built using **Python**, **Flask**, **Scikit-learn**, **XGBoost**, **HTML**, **CSS**, and **JavaScript**, the application features a modern AI-powered interface, interactive analytics dashboard, confidence score visualization, AI-based risk analysis, and a fully responsive design.

This project demonstrates the practical integration of **Machine Learning**, **Data Science**, and **Full Stack Web Development** to solve a real-world financial decision-making problem.
